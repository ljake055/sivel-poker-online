How private rooms work

1. Create a room and choose your table settings.
2. Share the invite link or four-character code.
3. Start once at least two players have joined.
4. Players can reconnect from the same browser if their connection drops.

Table rules

The host starts the table and deals each next hand. A 30-second turn timer automatically checks or folds inactive players.
